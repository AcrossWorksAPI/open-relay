# Superpowers Plans

Detailed implementation plans for this repository live in this folder.

Plans must include:

- goal
- constraints
- files likely to change
- acceptance criteria
- lifecycle/scope completeness pass
- assignment/scope matrix when relevant
- security/privacy risks
- test plan
- smoke plan
- closeout docs/matrix updates

Use checkbox syntax for executable steps. After a plan is implemented,
`docs/planning/PLAN_REGISTRY.md` must classify it as active, implemented,
superseded, dormant, or unknown.
